## Setup
Run the command:

    sudo ./setup.sh

## Images to use
Use the following images for this:
* quay.io/rlucente-se-jboss/myphp:latest
* registry.redhat.io/rhscl/mariadb-102-rhel7:10.2

## Run the containers
Run the command:

    ./start-app.sh

## Stop the containers
Run the command:

    ./stop-app.sh
